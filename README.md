# GameToDo - ゲーム性TODOアプリ

## 概要
タスク管理にゲーミフィケーションを組み込んだWebアプリです。
難易度・期限に応じたポイント獲得、称号システム、名刺カスタマイズ機能を実装しています。
大学の授業内チーム開発（6人・約2ヶ月）で作成し、16チーム中1位を獲得しました。

## 使用技術

- HTML / CSS / JavaScript
- GitHub（バージョン管理・チーム開発）

## 主な機能

### 1. タスク管理機能
- タスク名・難易度（5段階）・期限を設定して追加
- 難易度順に自動ソート表示
- 期限による色分け表示（黄色：期限3日以内、赤：期限超過）
- ホーム画面に「最優先タスク」（最も期限が近い未完了タスク）を自動表示
- 完了時に難易度×期限までの日数でポイント獲得
- タスク編集・削除機能

### 2. 時間割機能
- 曜日×コマ数の表形式UI
- セルクリックで科目名・教室を入力
- 登録した授業から直接タスク追加画面へ遷移
- 時間割リセット・編集機能

### 3. ゲーミフィケーション機能
- 実績システム：条件達成で称号獲得
- ショップ機能：獲得ポイントで称号購入
- 名刺機能：称号・背景色・模様・文字色のカスタマイズ、画像保存対応
